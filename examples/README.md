# SPL Student Environment

Welcome to your SPL course development environment! This workspace includes all the tools you need for course assignments.

## What's Included

- **C/C++**: GCC 13, Clang, CMake
- **Java**: Temurin JDK 21 + Maven
- **Python**: 3.12
- **SQLite**: Client and development libraries
- **VS Code Extensions**: C/C++, Java, Python, SQLite tools

## Quick Start

If you've completed the setup instructions, you're ready to go! Open a terminal in VS Code (`` Ctrl+` `` or `` Cmd+` ``) and verify everything works:

```bash
./run.sh
```

You should see "Hello, World!" output from C++, Java, and Python.
